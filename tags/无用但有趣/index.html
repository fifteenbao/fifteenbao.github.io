Jan 22, 2026

23年的双十一换了 Garmin 的 Forerunner265S

购买原因：

1. 可以不带手机跑步和听音乐 
2. 待机时间长 
3. 显得专业

想换《辐射》主题很久了…… 有看到喜欢的又没办法海外付款，最近试试用AI写了一个主题，目前还是比较满意的

Pip-Boy 3000 表盘 - Garmin Forerunner 265s

辐射（Fallout）系列游戏启发的Pip-Boy主题表盘，专为Garmin Forerunner 265s设计，采用标志性的绿色单色复古未来主义美学。

功能特性

  • 复古Pip-Boy设计：经典绿色单色配色方案 (#00FF00)
  • CRT屏幕效果：真实的扫描线叠加，呈现复古终端外观
  • 全面的数据显示：
    • 时间（时:分，带秒数）
    • 日期（日、月、年）
    • 步数计数器
    • 心率监测
    • 电池百分比
    • 消耗卡路里
  • 避难所男孩角色：标志性的辐射吉祥物
  • Pip-Boy边框：带角落支架的技术边框

安装说明

前置要求

  1. 安装 Connect IQ SDK Manager

    • 下载地址：https://developer.garmin.com/connect-iq/sdk/
    • 使用Garmin Connect账户登录

  2. 安装 Visual Studio Code

    • 下载地址：https://code.visualstudio.com/

  3. 安装 Monkey C 扩展

官方文档

  • 打开 VS Code
  • 进入扩展 (Cmd+Shift+X)
  • 搜索 “Monkey C”
  • 安装官方Garmin扩展

喝什么？

Jun 6, 2023

周日在湾厦参加了一个 《繁华与利剑》 策展人 陈嘉雯的讲座，说来搞笑，今天的展览，日历上记错了时间，昨天就来了……误打误撞听到了蛇口无车日 策展人吴文媛老师的演讲…… 有点意外之喜了。 来晚坐在最后面看工作人员的PPT一百多页……精美程度和质量堪称一流。整个讲解也超级精彩，一些历史背景、宗教文化、展陈思路之类的，活人感满满，也是想要写一下记录的原因。

“转译不是翻译”

这是后面圆桌讨论策展人的一个观点，做展览不是单纯的信雅达的“翻译”，而是有自己的理解思考，用自己文化视角的部分去做诠释理解消化。 让我想起了十几年前陕历博的导游说过，陕历博是唯一一个按照年代划分展厅的展馆，一种宏大历史叙事的感觉扑面而来。以至于过了这么多年，那个蹭别人导游团偷听到描述还记忆犹新。又想起了昨天吴文媛老师说到多元并不一定是美（虽然和这个没什么关系……），多视角的人物对于历史的描述，也算是历史迷人的一部分吧，不同的视角但都产生了美。

博物馆的策展人制度

也是圆桌讨论讲到的策展人制度。意外之喜，我爱深圳！

2023年，甘肃省文物局印发了《甘肃省各级博物馆策展人制度实施办法（试行）》，对策展人的选任、团队组建、职责、展览实施和结项评估等环节都做出来明确规定。甘肃出台的政策……也是蛮意外的。发现《深圳市南山区非国有博物馆扶持办法》会按照装修布展费用50%补贴，最高100万，南头古城补贴70%……还有按照参观人数核算的运行补贴，不愧是大南山。 又在小红书上搜了一下深圳有哪些策展人策展的活动……许知远、袁弘居然都在上面。 多元深圳！

历史笔记

展览还没去看，据说延期了，勤假还有0.9天，在考虑要不要请半天假去看展览。 先简单记一下

最贵的

贾汉基尔 （修泰姬陵那位的爸爸）的 ”命运之星“金币

细密画的 火、气、水、土 四大自然基本要素

火 - 气 - 水 - 地 = 天 - 风 - 水 - 地

火 = 神圣光明 气 = 天界精神 水 = 生命慈悲 土 = 物质人间

在画上的表现一般是 最上层：火/天（光明神圣） 第二层“气/风/天空（精神界） 第三层：水（生命、净化） 最下层：土/地（物质、承载）

昆丹工艺

一种珠宝加工工艺，类似于我们的景泰蓝掐丝珐琅之类的地位吧。 必看清单：

  • 摩羯手镯
  • 镶宝石盾牌

豆包帮我总结的：

  1. 找太阳纹/花卉纹中心
  2. 观察宝石边缘是否有珐琅彩绘
  3. 背面/侧面 是否有珐琅彩绘
  4. 区分盾钉（四角）和盾心宝石

连着两天下午在蛇口闲逛，新认识了好多有意思的人，以及好喜欢蛇口的氛围。
