我常用Sketch快捷键整理

Posted on 2017-02-27 Edited on 2019-07-25 In Design

设定一份属于自己的Sketch快捷键，能大大增加工作效率，当然了，也是很好的炫耀技能。
我的常用快捷键，有一些自定义的快捷键，大部分还是系统默认。

1.Zoom To All Selection And 100%

⌘+1/⌘+2/⌘+0

2.Jump From Artboard To Artboard

Fn+left/right

3.Bring To Front & Send To Back

⌃ +⌥ + ⌘+top/bottom

4.Copy And Paste Style

⌥ + ⌘+c

5.Undge Size

command+L/R/T/B

6.Lock Selection On Move

⌥+⌘+mouse~

7.Contained Selections

⌥+mouse~

8.Drag Pasted Copies

sketch自带快捷键

窗口

⌥ + ⌘ + 1 显示 / 隐藏图层列表 – 左边栏
⌥ + ⌘ + 2 显示 / 隐藏图层列表 – 右边栏
⌥ + ⌘ + 3 显示 / 隐藏图层列表 – 左右边栏
⌥ + ⌘ + T 显示 / 隐藏工具栏
⌘ + . 演示模式
⌃ + ⌘ + F 全屏模式

字体

command ⌘ + B 粗体
command ⌘ + I 斜体
command ⌘ + U 下划线
⌥ + ⌘ + + 增大字体（Increase Font Size）
⌥ + ⌘ + – 减小字体（Decrease Font Size）
⌥ + ⌃ + L 增加字符间距
⌥ + ⌃ + T 减少字符间距
⇧ + ⌘ + O 将文本转换为轮廓
⇧ + ⌘ +[/]/\文字左／右／中
⌃ + ⌘ + space 特殊字符

图形编辑

⌃ + ⌘ + M 使用蒙版
⌥ + ⌘ + U 合并形状
⌥ + ⌘ + S 减去上层形状
⌥ + ⌘ + I 截取重叠区域
⌥ + ⌘ + X 合并及减去重叠区域
⌃ + C 拾色器

sketch自定义快捷键

⚠️区分大小写

  • ⌘ + ⇧ + W
    Arrange->Align->Top 上对齐

  • [] ⌘ + ⇧ + A
    Arrange->Align->Left 左对齐

  • ⌘ + ⇧ + D
    Arrange->Align->Right 右对齐
  • ⌘ + ⇧ + B
    Arrange->Align->Bottom 下对齐
  • ⌘ + ⇧ + H
    Arrange->Aligne->Horizontally 水平居中
  • ⌘ + ⇧ + V
    Arrange->Align->Vertically 垂直居中
  • ⌃ + ⌘ + ⇧ + H
    Arrange->Distribute->Horizontally 水平等距
  • ⌃ + ⌘ + ⇧ + V
    Arrange->Distribute->Vertically 垂直等距

sketch插件快捷键

AEFlowchart流程图插件

⌃+⇧+s start/end
⌃+⇧+p process
⌃+⇧+d decision
⌃+⇧+o input/output

distributor等间距插件

⌃+alt+d 调出窗口
⌃+alt+a 重复上一次

find and replace查找替换

⇧ + ⌘ + f

Font Finder 在当前页面中查找和选择字体项目

⌘ + ⌃ + ⇧ + F

以上为常用的快捷键，在B站分享了几个插件，
