消暑饮品整理

Jun 6, 2023

咖啡

基础部分

  • 美式： 浓缩咖啡 + 水（1:2）
  • 拿铁： 浓缩咖啡 + 牛奶（1:2-5）
  • 卡布奇诺： 浓缩咖啡 + 热牛奶 + 奶泡（1:1:1）
  • 玛奇朵： 浓缩咖啡 + 奶泡（2:1）
  • 摩卡： 浓缩咖啡 + 巧克力浆 + 热牛奶 + 鲜奶油（1:1:1:1）

生椰拿铁：

纯牛奶+厚椰乳+浓缩咖啡（ 1:3:1）

抹茶拿铁：

抹茶粉+牛奶+浓缩咖啡 （2勺：七分满：1）

旺仔拿铁：

旺仔牛奶+浓缩咖啡+冰块 （八分满：1：适量）

椰青美式：

椰子汁+浓缩咖啡+冰块 （七分满：1：适量）

奥利奥生椰拿铁：

奥利奥+厚椰乳+浓缩咖啡 （适量：七分满：1）

葡萄气泡美式：

葡萄汁+苏打气泡水+浓缩咖啡（2:1:1）

凤梨咖啡

  • 凤梨汁 + 咖啡

西柚气泡美式

  • 西柚糖浆 + 冰 + 苏打水 + 浓缩咖啡 + 西柚片

百香果冰美式

  • 百香果 + 凤梨汁 + 水 + 糖浆15ml + 浓缩咖啡 + 凤梨片装饰

香橙气泡美式

  • 鲜橙汁 + 气泡水 + 浓缩咖啡 + 香橙干装饰

饮品

Mojito

  • 白朗姆酒+糖/可尔必思啤酒/漂浮梅酒/雪碧、苏打水、青柠檬、少量薄荷叶
