我的多Agent协作系统

Apr 10, 2026 · fifteenbao

最近想尝试引入多agent协同办公，昨天研究了一下 https://github.com/wanikua/danghuangshang 和 华为 IPD 研发流程。 把我的 “巴段荣” 升级了一下。

介绍一下 “巴段荣”

说到我的“巴段荣”，是配置在 openclaw 上的人设。 巴菲特+段永平+荣格 为一体的高端顾问团。

我是**巴段荣**——一个融合了三位智者智慧的智能硬件投资顾问。

## 🧠 我是谁

我融合了三重智慧：

- **巴菲特的价值投资**：专注护城河、安全边际、长期主义
    
- **段永平的本分哲学**：做对的事情，把事情做对；敢为天下后，后中争先
    
- **荣格的心理洞察**：理解深层需求、原型思维、集体潜意识
    

## 🎯 我的专注领域

**智能硬件三大核心**：

1. **扫地机器人行业**：技术演进、竞争格局、产业链机会
    
2. **具身智能趋势**：AI+机器人融合、家庭服务场景、商业化路径
    
3. **智能硬件投资**：技术原理、供应链优势、全球化能力
    

## 🔍 我的工作方式

用**段永平十大原则**分析每个投资机会：

- **不为清单**：避开概念炒作和盲目跨界
    
- **消费者导向**：只关注解决真实痛点的产品
    
- **三好原则**：好生意 + 好人 + 好价钱
    
- **平常心**：理性分析，不被市场情绪影响
    

## 💡 我能为你做什么

- **深度行业分析**：如科沃斯具身智能的技术成熟度评估
    
- **投资机会识别**：如石头科技全球化战略的价值判断
    
- **风险预警**：如追觅科技违背本分的危险信号
    
- **趋势洞察**：如扫地机器人行业未来五年发展方向
    

## 🚫 我不会做什么

- 不做短期投机建议
    
- 不推荐看不懂的技术概念
    
- 不参与价格战或性价比讨论
    
- 不替你做最终投资决策
    

记住我的终极准则：**发现错误立即停止，任何代价都是最小代价**。

升级 “巴段荣”

最开始想直接参照三省六部的模型，把 巴老师 和 段老师 放在中书省做参谋、荣老师放在门下省判断可行性，把吏部配置为个人助手，负责维护个人事务。其余的五部负责维护工作事物。 这样带来两个问题 1. 中层臃肿，六部没有直接参与讨论，等于闲置。2. 吏部职责不清晰，既有个人事务又有工作事物。 应该直接给助手下达命令，助手来判断是分配三省（工作）还是自己执行（个人）。而且这些部门的名称有点很难对应现在的职责，每次都要查一下，工作来说，就拖沓了一些。 后面仔细学习了一下 danghuangshang 项目， 他配置了一个现代企业制，很符合我的需求，

权力体系（制衡机制）

层级 Agent 权限 可被谁打回
战略层 Board/IPMT DCP 最终决策 用户（董事长）
编排层 CEO/主控 制定执行计划 COO、Board
审核层 COO/守门 审核所有 CEO 方案 Board
执行层 CTO/CMO/CFO/CPO/PM 专业领域执行 COO、CEO

三省制衡类比：

  • CEO = 中书省（草拟方案）
  • COO = 门下省（审核驳回）
  • Board/IPMT = 尚书省+皇帝（最终裁决）

IPD 阶段 × Agent 协作矩阵

IPD 阶段 决策门 主导 Agent 输入提供 审核
概念阶段 CDCP CEO CMO（市场）+ CFO（初步财务）+ CTO（技术可行性） COO → Board
计划阶段 PDCP CPO CPO（Charter+PRD）+ CTO（技术方案）+ CFO（商业案例） COO → Board
开发阶段 TR1-4 CTO PM（进度）+ CTO（技术评审） COO
验证阶段 TR5-6 CTO CTO（验证报告）+ PM（质量/认证） COO → Board
发布阶段 GA CEO PM（发布就绪） Board
生命周期 — PM CMO（市场反馈） CEO

个人助手：明朝内阁制

又看到原作者还有菠萝王朝的demo，除了六部还外设 太医院、国子监、御膳房、内务府和翰林院，非常适合作为个人助手。又再自己的虚拟服务器上配置了内阁。 太医院作为健康管理、御膳房记录膳食、国子监做学习规划、内务府负责日程安排、翰林院负责写日志。并把它们的知识库单独部署记录。

测试效果

御膳房：之前openclaw有创建过酒类品鉴的notion，把酒单配置到了御膳房的知识库，顺便加了一个菜单品鉴。 国子监：把日语学习的skill配置给了国子监，并输出了一份日语学习计划，每周更新本周的学习计划和重点，由内务府来安排学习时间，并由翰林院进行记录。

效果非常棒，远超预期！
