Framer X正式版

发表于 2018-09-23 | 分类于 Design - FramerX

正式版上线增加了令人欣喜的在线商店，现在内容比较少，未来的扩展空间非常大。新版降低了对于代码的使用门槛，可以让更多有代码基础的设计师使用，交互更加真实。有听说和Sketch兼容很好，但Symblo还没有支持，需要拆开复制。喜欢Stack。
