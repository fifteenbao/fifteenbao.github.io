Sketch版本管理

发表于 2018-10-16 | 分类于 Design

看到GitHub上有开源插件，按照教程走了一波。

相关准备

  • 首先，你需要在你的任意 Mac 上安装在 Sketch 官网安装的 Sketch
  • 第二，下载好 Sketch Git 插件
  • 第三，下载好最新版的 Git

Sketch Git & Git 的安装

安装git.sketchplugin

安装git-2.6.4-intel-universal-mavericks.dmg

Gitlab 注册 & SSH Key 获取

服务器上的 Git - 生成 SSH 公钥
Mac终端输入 “

your_email 处填写你自己邮箱

复制SSH

粘贴SSH到GitLab

SketchTool 的安装(这里有问题)

似乎找不到 install.sh 文件，可能是这个原因导致push出错，后面换了终端上传。

项目的建立 & 克隆到本地

在GitLab Project 并设置visibility，点击 Creat project。
建立成功后会有几行代码直接复制Create a new repository的代码到终端，

Create a new repository

url 为gitlab上地址
file 文件夹名字

这个步骤可以看到生成README.md文件

Sketch链接到Git

Sketch 设计稿拷贝到拉取的文件夹，打开Sketch，点击Plugins - Git - Add File to git
将Sketch链接到Git

这个时候 Sketch 下方会出现 File added to git 就表示这个文件已经被追踪。

修改sketch后，点击 Plugins - Git - Commit. 提交修改记录。
点击Plugins - Git - Push推送到远端服务器

Push 会报错，选择用终端上传
